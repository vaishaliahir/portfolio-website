My Projects

WorkFlex

WorkFlex is a full-stack freelance hiring platform developed using Django and MySQL that connects clients with both technical and non-technical professionals. Users can create and manage professional portfolios showcasing their skills, services, and experience. Clients can explore freelancer profiles, compare portfolios, and hire suitable candidates based on their requirements. The platform also includes secure payment gateway integration for smooth online transactions and a user-friendly responsive interface.

AI Examiner

AI Examiner is an AI-powered answer evaluation system developed using Django, MySQL, and Gemini API. The platform allows teachers to securely log in and upload students’ answer sheets for automated evaluation. Using AI-based analysis through Gemini API integration, the system checks answers, evaluates performance, and generates scores automatically. The project was designed to reduce manual checking effort, improve evaluation efficiency, and provide fast and accurate result analysis through an intelligent automated workflow.

Car Rental System

Car Rental System is a desktop-based application developed using JavaFX for managing car rental services efficiently. The system allows users to browse available cars, select rental options, and book vehicles based on their requirements. It also includes insurance features such as Premium and Normal insurance plans for enhanced customer safety and flexibility. A payment interface was designed to simulate the booking and payment process, providing a complete end-to-end car rental experience with a user-friendly graphical interface.
